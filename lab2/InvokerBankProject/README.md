# Bank Invoker Game (HW2 Solution)

Это решение домашнего задания, сочетающее банковский процессинг (SOLID, паттерны, многопоточность) и GUI в стиле Dota 2 Invoker.

## Структура проекта
* **model**: Сущности (User, Account, Transaction)
* **patterns**: Реализация паттернов (Strategy, Factory, Observer)
* **service**: BankService (Singleton, ExecutorService для асинхронности)
* **game**: JavaFX приложение (InvokerEngine, UI)

## Требования
1. **Java 17** или выше.
2. **Maven** (установленный в системе).

## Как запустить

### 1. Сборка и запуск через Maven
Просто выполните команду в корне проекта:

```bash
mvn clean javafx:run
```

### 2. Управление
Когда откроется окно:
1. Нажимайте **Q, W, E** на клавиатуре, чтобы добавлять сферы (как в Доте).
2. Нажмите **R**, чтобы скастовать заклинание (выполнить транзакцию).
3. Следите за балансом в центре экрана.
